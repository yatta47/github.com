# 概要

hugoを使ってサイト更新をする際に必要な情報

## コマンドライン

必要なコマンドライン

### 記事を作成する

```command
hugo new post/[dir-name]/[article-name].md
```

| Name         | Description    |
| ------------ | -------------- |
| dir-name     | ディレクトリ名 |
| article-name | 記事名         |

### テーブルを整形するショートカットキー

Alt + Shift + F

### レベルを上げる

Ctrl + Shift + [ or ]
